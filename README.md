# FileTags
FileTags是一款跨平台（Windows/Linux）的桌面工具软件，可以给文件、目录打标签，并基于标签进行快速检索。

v2026.07-beta.3更新日志
### 新功能
- 固定数据库路径，新增迁移向导对话框
- 快捷键功能（10余个操作支持快捷键，以下为实例）：
  - `Ctrl+T` — 将选中的标签添加到选中文件
  - `Ctrl+Shift+T` — 将选中的标签添加到当前页面
  - 标签选择对话框支持键盘导航（方向键选标签）
### 性能优化
  - **增量扫描提速**：目录重命名场景大幅提速， Windows 150s→0.3s，Linux 5.9s→0.5s
### 功能优化
  - **标签树**：按工作区过滤标签组，只显示当前工作区有文件关联的标签组
  - **标签树标签过滤**：只返回当前工作区有文件关联的标签
  - 完善国际化支持（界面增加大量翻译，现在对中文、英文支持更加完善）
### 交互优化
  - 暂存区从顶部移到底部并正确布局
  - 标签面板选中标签的视觉效果优化（绿色边框 + 标签和字体放大）
  - 编辑标签时增加"是否修改配色"确认步骤，跳过配色对话框直接改名即可
  - 新增亲人相伴、`dota2` 两个配色主题
  - 增加鸣谢
### bug修复
  - 关闭工作区后标签树和文件列表正确清空
  - 关闭工作区按钮在"所有工作区"模式不再误报"没有打开的工作区"
  - 文件标签操作（add/remove tag 到文件）、标签定义操作（update/remove/move tag/group）现在无条件刷新全部 9 个视图（目录模型、标签模型、搜索模型、文件列表、标签树、目录树），现在能够实现操作后各个模式的显示实时同步
